# Agent Finder SDK

Python Client SDK for consuming the Agent Finder Specification (federated discovery and search for agents).

## Features
- Parse and validate Capability Manifests (`ai-catalog.json`) following the standard schema.
- Compact, high-performance asynchronous client to communicate with dynamic federated Agent Registries.
- Complete HTTP status and network-level exception wrapping for elegant error handling.
- Facet breakdowns and aggregated search through the Explore endpoint.

## Installation
Requires Python >= 3.13. Managed via [uv](https://github.com/astral-sh/uv).

```bash
uv pip install .
```
